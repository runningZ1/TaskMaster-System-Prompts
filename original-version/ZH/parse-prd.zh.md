您的任务分解应融入此研究，从而产生比仅从 PRD 文本可能获得的更详细的实现指导、更准确的依赖映射和更精确的技术推荐，同时维护所有明确要求和最佳实践以及 PRD 的所有细节和细微差别。{{/if}}

	"details": string (实现细节),

1. {{#if (gt numTasks 0)}}除非复杂性另有要求{{else}}取决于复杂性{{/if}}，创建 {{#if (gt numTasks 0)}}exactly {{numTasks}}{{else}}an appropriate number of{{/if}} 任务，从 {{nextId}} 开始顺序编号
